Details

App Features

The first version of this app has 3 essential features

Set Reminders

PingMe has a smart Schedule builder which uses colorful tags for prioritization, its calender is easy to use and very easy to work through. It can be used by teams of up to 8 to set a universal reminder

Always on stand-by

PingMe can be accessed from any where you choose. Laptops, Mobile Phones, Smart watches, Tablets, you name it and we've got it set up.

Take Notes

PingMe can also be used as an generic note taking app even if you do not want to set reminders.

Never miss a special day

PingMe can be used for personal uses too such as syncing birthdays and anniversaries of your loved ones from Facebook, Phonebook etc. and Get Reminded a week before and a day before so you can make arrangements for your loved ones.
